

## Project info
This project is a simple web application that allows users to manage their finances. It uses React for the frontend, TypeScript for type safety, and shadcn-ui for UI components. The project is built with Vite, which is a modern build tool that is fast and easy to use. It also uses Tailwind CSS for styling, which is a utility-first CSS framework that is easy to use and highly customizable. The project is also tested with Jest and Vitest, which are popular testing frameworks for JavaScript and TypeScript projects. The project is also linted with ESLint and Prettier, which are popular code linting and formatting tools for JavaScript and TypeScript projects.
